# Escenario 2 — Detección de Anomalías en Strings Fotovoltaicos

Detección de anomalías en 2,500 strings de un parque solar (tiempo real: discriminación nube vs. fallo; e histórico) sin agotar la memoria del sistema. Este escenario se resuelve a nivel de **diseño** en el documento técnico (Seccion 4); esta carpeta contiene los manuales Modbus/SmartLogger y las estimaciones de carga (`datasheets/`).
