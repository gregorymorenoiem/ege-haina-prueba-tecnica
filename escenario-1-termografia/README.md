# Escenario 1 — Termografía Predictiva

Análisis acelerado con IA de imágenes termográficas (cámaras fijas, portátiles y dron) para detección de hot-spots y clasificación de defectos. Este escenario se resuelve a nivel de **diseño** en el documento técnico (Seccion 3); esta carpeta contiene los datasheets y manuales de los equipos seleccionados (`datasheets/`).
